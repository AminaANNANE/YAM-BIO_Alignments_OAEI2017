# YAM-BIO_Alignments_OAEI2017
The Zip file contains:

      1.The alignments produced by YAM-BIO system during the ontology alignment evaluation initiative (OAEI 2017) for the six tasks of large biomedical ontologies track
      
      2. The alignment produced by YAM-BIO system during OAEI 2017 for Anatomy track
      
      3.The file used as background knowledge that contains existing mappings extracted from UBERON and DOID ontologies
